📌 Team Task Manager
🧾 Project Overview

The Team Task Manager is a full-stack web application designed to simplify task management within teams. It allows users to create tasks, assign them to team members, and monitor their progress in a clear and organized way.
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
✨ Key Features

🔐 User authentication (Signup & Login)
📝 Create and assign tasks to team members
📊 Track task status in real time (Pending, In Progress, Done)
------------------------------------------------------------------------------------------------------------------
🛠 Tech Stack Used
⚡ FastAPI – Backend framework for APIs
🎨 Streamlit – Frontend interface
🗄️ SQLite – Lightweight database for storage
-------------------------------------------------------------------------------------------------------------------
▶️ How to Run the Project Locally
🔹 Backend Setup
cd backend
uvicorn main:app --reload
-------------------------------------------------------------------------------------------------------------------
🔹 Frontend Setup
cd frontend
streamlit run app.py
